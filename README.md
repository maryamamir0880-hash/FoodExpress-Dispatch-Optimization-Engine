# FoodExpress-Dispatch-Optimization-Engine
FoodExpress Dispatch Optimization Engine is a C++ based food delivery management system that uses Data Structures and Algorithms including Heap, Queue, Stack, BST, Hash Table, Linked List, Graph (Dijkstra), and Merge Sort for order scheduling, rider dispatching, route optimization, and performance analysis.
Features

Dynamic Order Scheduling using Priority Heap

Kitchen Load Analysis using Queues

Rider Dispatch Optimization

Route Optimization using Dijkstra Algorithm

Search & Retrieval using Hash Tables and BST

Order History Tracking using Linked Lists

Undo Operations using Stack

Scalability and Performance Analysis

System Reporting Module

Data Structures Used

Priority Heap

Queue

Stack

Linked List

Binary Search Tree (BST)

Hash Table

Graph (Dijkstra Algorithm)

Merge Sort

Language

C++

Purpose

This project was developed to demonstrate the practical implementation of Data Structures and Algorithms in a real-world food delivery management system.
